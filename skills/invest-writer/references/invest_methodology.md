# 投资分析方法论

## 一、行业分析框架

### 1. 行业规模与增长空间

**关键问题**：
- 这个行业有多大？（市场规模）
- 还能长多大？（增长空间）
- 长得有多快？（增长率）

**数据来源**：
- 研报（券商研报、咨询公司报告）
- 行业协会数据
- 政府统计数据
- 公司财报中的行业分析

**呈现方式**：
```
根据QYR（恒州博智）的统计，2023年全球电机市场规模达到1501.6亿美元，预计到2030年将增长至2380亿美元，年复合增长率（CAGR）为6.9%。
```

### 2. 行业发展阶段

**四阶段模型**：

| 阶段 | 特征 | 投资策略 |
|------|------|----------|
| 初创期 | 市场小、增长快、竞争少、亏损多 | 高风险高收益，适合风险投资 |
| 成长期 | 市场扩大、竞争加剧、利润提升 | 最佳投资期，寻找龙头 |
| 成熟期 | 增长放缓、竞争格局稳定、现金流好 | 寻找被低估的龙头 |
| 衰退期 | 市场萎缩、竞争减少、利润下降 | 谨慎投资，关注转型 |

**判断标准**：
- 市场渗透率（5%-30%为成长期黄金区间）
- 行业增速（>20%为高增长）
- 龙头企业毛利率（>20%说明有定价权）

### 3. 竞争格局

**分析维度**：
- 市场集中度（CR3、CR5）
- 主要玩家及市场份额
- 进入壁垒
- 替代品威胁

### 4. 行业打分标准

**满分10分，参考以下维度**：

| 维度 | 高分标准 | 低分标准 |
|------|----------|----------|
| 增长空间 | 5年CAGR > 15% | 5年CAGR < 5% |
| 竞争格局 | 龙头优势明显 | 竞争激烈无龙头 |
| 政策支持 | 政策强力支持 | 政策限制或打压 |
| 技术壁垒 | 技术壁垒高 | 技术壁垒低 |
| 周期属性 | 弱周期 | 强周期 |

## 二、企业分析框架

### 1. 商业模式分析

**核心问题**：这家公司是怎么赚钱的？

**分析框架**：
- 客户是谁？
- 提供什么产品/服务？
- 收入来源是什么？
- 成本结构是什么？
- 核心竞争力是什么？

**比喻类比法**：
当商业模式复杂时，用生活中的类比来解释：
```
这就像两个小年轻，3个月后要结婚了，现在要结婚前把婚房装修好。你当然可以，自己一步一步来，自己砸墙，自己刮大白，自己贴瓷砖；或者你也可以找个装修队，直接外包，到时候自己买点家具就可以直接入住了。

药明康德就是这个装修队，提供一整套服务。
```

### 2. 护城河分析

**巴菲特护城河四要素**：

| 护城河类型 | 含义 | 案例 |
|------------|------|------|
| 无形资产 | 品牌、专利、牌照 | 茅台的品牌、药明康德的分子库 |
| 转换成本 | 客户换供应商的成本高 | 企业软件、银行账户 |
| 网络效应 | 用户越多价值越高 | 微信、淘宝 |
| 成本优势 | 规模经济或独特资源 | 海螺水泥的区位优势 |

**判断标准**：
- 是否有定价权？（能否涨价而不流失客户）
- 是否有高客户粘性？（老客户续约率）
- 是否有规模优势？（成本随规模下降）

### 3. 管理层分析

**关注点**：
- 创始人背景
- 管理层持股比例
- 过往业绩
- 股权激励方案
- 信息披露质量

### 4. 企业打分标准

**满分10分，参考以下维度**：

| 维度 | 高分标准 | 低分标准 |
|------|----------|----------|
| 商业模式 | 清晰、可持续 | 模糊、依赖单一客户 |
| 护城河 | 明显且宽阔 | 无明显护城河 |
| 管理层 | 诚信、能干 | 有污点或能力存疑 |
| 行业地位 | 龙头或前三 | 中游或下游 |

## 三、财务分析框架

### 1. 成长性分析

**关键指标**：
- 营收增长率
- 净利润增长率
- 营收增长的来源（量增还是价增）

**判断标准**：
- 连续3年营收增长 > 15% → 高成长
- 连续3年营收增长 5%-15% → 稳健成长
- 营收增长 < 5% 或负增长 → 成长性不足

### 2. 盈利能力分析

**关键指标**：
- 毛利率（反映产品竞争力）
- 净利率（反映整体盈利能力）
- ROE（净资产收益率）

**判断标准**：
- 毛利率 > 40% → 强定价权
- 净利率 > 15% → 盈利能力强
- ROE > 15% → 优秀

### 3. 财务健康度分析

**关键指标**：
- 资产负债率（< 60%为健康）
- 经营现金流（> 净利润为佳）
- 应收账款周转天数
- 存货周转天数

### 4. 财务打分标准

**满分10分，参考以下维度**：

| 维度 | 高分标准 | 低分标准 |
|------|----------|----------|
| 成长性 | 营收增速 > 15% | 营收增速 < 5% |
| 盈利能力 | ROE > 15% | ROE < 10% |
| 财务健康 | 负债率 < 50% | 负债率 > 70% |
| 现金流 | 经营现金流 > 净利润 | 经营现金流 < 净利润 |

## 四、估值方法

### 1. 相对估值法

**常用指标**：
- PE（市盈率）
- PB（市净率）
- PS（市销率）
- PEG（市盈率相对盈利增长比率）

**使用场景**：
- PE：适用于盈利稳定的公司
- PB：适用于资产重的公司（银行、地产）
- PS：适用于亏损或微利的成长公司
- PEG：适用于高增长公司（PEG < 1 为低估）

### 2. 绝对估值法

**DCF（现金流折现法）**：
- 预测未来10年自由现金流
- 折现到当前
- 计算企业价值

**注意**：DCF对假设敏感，需要做敏感性分析。

### 3. 估值判断

**高估信号**：
- PE远高于行业平均
- PE远高于历史平均
- PEG > 2
- 股价已反映未来3-5年的增长预期

**低估信号**：
- PE低于行业平均
- PE低于历史平均
- PEG < 1
- 股价未反映合理的增长预期

## 五、风险分析框架

### 1. 行业风险

- 政策风险
- 技术变革风险
- 周期性风险
- 竞争加剧风险

### 2. 公司风险

- 管理层风险
- 财务风险
- 业务风险（单一客户依赖、单一产品依赖）
- 法律风险

### 3. 估值风险

- 市场情绪变化
- 利率变化
- 流动性风险

## 六、投资决策框架

### 1. 买入标准

**必须满足的条件**：
- 行业有增长空间
- 公司有护城河
- 财务健康
- 估值合理或低估
- 自己能看懂

**加分项**：
- 管理层优秀
- 有催化剂（政策、新产品、业绩拐点）
- 自己有认知优势

### 2. 卖出标准

**应该卖出的情况**：
- 发现当初的判断错了
- 公司基本面恶化
- 估值明显高估
- 有更好的投资机会
- 需要用钱

**不应该卖出的情况**：
- 股价跌了就卖（除非基本面恶化）
- 股价涨了一点就卖（容易卖飞）
- 看到别人卖就卖

### 3. 持仓管理

**分散原则**：
- 单只股票不超过总仓位的20%
- 同一行业不超过总仓位的40%
- 5-10只股票为宜

**长期持有**：
- 看得懂的好公司，长期持有
- 定期复盘，但不要频繁交易
- 记录自己的投资逻辑，定期检验

## 七、写作时的数据呈现

### 1. 数据来源标注

```
根据QYR（恒州博智）的统计...
根据公司财报...
根据券商研报...
```

### 2. 数据对比

```
营收增速不快，利润大幅增长，那么钱大概率是省下来的。
```

### 3. 数据解读

不要只列数据，要解读：
```
最近几年营收增长并不快，都是个位数的增长，利润自2023年触底之后，开始了连续两年的反弹，毛利率，净利率相比前几年也有上升，可见企业正在慢慢的走出低谷。
```

## 八、常见投资陷阱

### 1. 价值陷阱

看似便宜，实则陷阱：
- 行业在衰退
- 竞争力在下降
- 财务造假风险

### 2. 成长陷阱

看似高增长，实则不可持续：
- 增长靠并购
- 增长靠降价
- 增长不可持续

### 3. 热点陷阱

追热点，买在高点：
- 市场情绪高涨时买入
- 不了解公司基本面
- 没有估值概念

## 九、投资文章的底线

### 1. 不做投资建议

可以表达自己的判断和决策，但不能建议读者买入或卖出。

### 2. 披露持仓

如果自己持有或计划买入相关股票，应该披露。

### 3. 风险提示

每篇文章都要有风险提示，提醒读者投资有风险。

### 4. 不预测短期

关注长期价值，不预测短期股价走势。
